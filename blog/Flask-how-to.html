Flask How to

Install Virtual Env (one time)

$ sudo easy_install virtualenv

Installing Flask (Do it project by project rather than system-wide)

Let's say we are doing a project called "HelloWorld"

$ mkdir HelloWorld
$ cd HelloWorld
$ virtualenv venv
$ . venv/bin/activate

The first command makes a new project folder. The next takes you into the project folder. The virtualenv venv command creates a virtual environment sub-folder called "venv" in this project folder. The . venv/bin/activate starts or activates the virtual environment.

Next we have to activate Flask in the virtual env:

$ sudo pip install Flask

This command installs flask into the venv/bin/pip/ folder. Don't really need the sudo

Creating and running a Minimal Flask Application

Open a new document and type in the following code and save file as "manage.py"

```python

Import an instance of the Flask class. This is the web server gateway interface (WSGI) application

from flask import Flask

Create instance of the class

app = Flask(name)

@app.route('/') def hello_world(): return 'Hello World!'

if name == 'main': app.run() ```

Then run the code inside your environment

$ python manage.py

You will see the following message:

* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)

Now, you can open up a browser and access the webpage from http://127.0.0.1:5000
